# Changelog

This workspace does not include `.git` history, so older changes could not be reconstructed.
Entries below start from the current working copy.

## 2026-03-11

- Fixed the Isaac Sim interactive loop so the app follows the Kit lifecycle instead of running an unconditional `while True`.
- Added richer visual profiles for recyclables: bottles, jugs, cans, cartons, paper stacks, bags, and glass/residue clusters.
- Expanded the demo item catalog with `aseptic_carton`, `detergent_jug`, and `glass_shard_cluster`.
- Added a GUI viewport option so Isaac Sim can look through the RGB camera directly.
- Added a visible camera rig in the scene and updated the docs with camera-view instructions.
